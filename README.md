## Формула оценки

_Накопленная оценка_ = _0.5_ $\times$ _ДЗ1_ + _0.5_ $\times$ _ДЗ2_

_Итоговая оценка_ = _0.6_ $\times$ _Накопленная оценка_ + _0.4_ $\times$ _Экзамен_
